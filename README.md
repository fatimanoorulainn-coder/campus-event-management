## 🚀 Overview

The **Campus Event Management System** is a comprehensive web platform designed to streamline event management in university environments. It provides:

- **Students** - Browse, search, and register for events
- **Administrators** - Create, manage, and monitor events
- **Real-time updates** - Instant registration confirmations and notifications
- **Modern UI** - Glassmorphism design with smooth animations

## 🛠️ Technology Stack

### Backend
- **Java 11+** - Core language
- **JSP/Servlets** - Web layer
- **JSTL** - JSP tag library
- **MySQL 8.0** - Database
- **JDBC** - Database connectivity
- **BCrypt** - Password hashing
- **Apache Commons DBCP** - Connection pooling
- **Log4j** - Logging
- **Gson** - JSON processing

### Frontend
- **Bootstrap 5.3** - UI framework
- **Font Awesome 6** - Icons
- **CSS3** - Custom styles (Glassmorphism, Animations)
- **JavaScript** - Interactivity, Form validation
- **HTML5** - Markup

### Build & Deployment
- **Maven** - Build tool
- **Apache Tomcat 9+** - Servlet container
- **Git** - Version control
